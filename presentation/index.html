SO WHAT IS THE MANDELBAUM?

🌳

In 1980, a man named Benoît Mandelbrot looked at the equation z → z² + c and thought:

"what if I just... loop this forever"

(he was a genius. we are just vibing.)

THE TERRIFYING FORMULA

zn+1 = zn² + c where: z = complex number (scary) c = also complex number (also scary) n = how many times you repeat it ∞ = when you give up

If |z| stays ≤ 2 forever → point is IN the set (black)
If it escapes to ∞ → it's OUT → colour it based on how fast it ran away

That's literally it. The whole thing. I promise.

NOW WHAT IS KUBERNETES?

🤔

Imagine you have a bunch of computers.
Now imagine you need something to babysit all those computers.
Now imagine the babysitter is also made of computers.

That's Kubernetes.

Greek for "helmsman". The logo is a steering wheel.
Everyone calls it k8s because typing is hard.
